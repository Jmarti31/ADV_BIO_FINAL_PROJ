How does it work?

The following browsing feature of the database retrieves miRNA/Microbiota interactions associated
with a given query. The MicroRNA option uses exact miRBase ID's for search queries, while the
Microbiota option uses primarily exact NCBI Organism Name's and has limited functionality
to use NCBI Taxonomy ID's also as an option. The server is designed to reset the page if there
doesn't appear to be any associated data to a given query.
